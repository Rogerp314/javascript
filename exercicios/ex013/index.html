A diferença entre variavel simples e composta

O proprio nome ja diz oq significa, sendo assim a variavel simples so pode armazernar um valor por vez, enquanto a variavel composta pode armazernar varios valores numa mesma estrutura.

A variavel composta e como se fosse varias variaveis simples num bloco so, sendo assim para indentificar elas usa-se [] e o nome da variavel simples dentro da composta.
Exemplo:

        let a = [variavel 0, varavel 1, varavel 2... varavel n]

Desta forma podemos ver que a variavel composta organiza os valores de tal forma que seja respectivo a ordem colocada entre colchetes, assim cada valor recebe o nome da variavel composta e depois o indice dela.
Obs: Os indices (o numero de identificaçao) começa no 0
Uma variavel composta como esta pode ter quantos espaços precisar e ela se chama array ou vetor em portugues. O array e formado por elementos sendo eles as proprias variaveis do exemplo, e cada elemento tem seu conteudo, chave (numero de indentificaçao)e ele ocupa um determinado espaço na memoria. Cada array tem seu elemento com sua determinada chave e conteudo (valor).

Para acresentar valores no vetor (variavel composta) sem perder os valores originais e sem criar outro vetor usa-se o comando:

        (nome do vetor sem os parenteses; ex:) num[posiçao em que eu quero colocar o novo valor]= valor do novo elemento

Esse exemplo serve para tudo tipo eu posso escolher onde colocar o valor do elemento escolhendo o valor do indice entre os colchetes depois do nome do vetor e antes do sinal de igual, mas se eu so quiser colocar o elemento no ultimo indice sem escolher manualmente e so usar o comando num.push(valor do elemento ex: 4) assim cria-se um elemento no ultimo indice com o valor colocado entre parenteses.

Para eu saber o comprimento do meu vetor, ou melhor quantos elementos tem o meu vetor usa-se o comando num.length.
Se eu quiser os valores do meu vetor de forma crescente sem precisar eu mesmo organizar usa-se o comando num.sort() assim os valores ficam na ordem crescente

Obs: Se na hora de colocar os valores nos idices e eu escolher um indice ja ocupado o novo valor atribuido a ele substituira o valor anterior.
Sempre que eu for atribuir um atributo para qualquer variavel e quiser mostrar devo escrever (nome do variavel).atributo

Se eu quiser posso mostrar um elemento isoladamente usando o comando num[indice que quero mostrar]

Para eu conseguir mostrar todos os elementos do vetor de forma simples sem precisar ficar repetindo os comandos eu somente mudo o indice ex: num[indice qualquer] no indice posso colocar uma variavel que começa a onde eu quero e termina a onde eu quero usando as tecnicas:

  • While;
  • do while;
  • for

Sendo assim eu so faço uma estrutura de repetiçao e coloco a variavel no colchetes do vetor.
