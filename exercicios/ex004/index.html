O DOM serve para vc localizar qualquer elementos no site facilitando como colocar objetos e elementos no site. Sendo que o numero entre couchetes e o elemento em si, tipo se temos mais de um elemento igual temos que localiza-lo e a forma e depois de colocar a classe do elemento nos parenteses na variavel colocamos entre couchetes o numero ordenado do elemento menos 1.
Esse numero serve para indicar quando usamos o comando getElementsByTagName pois esse comando engloba todas a tags que colocamos entre parenteses, ou seja, sempre que tiver comandos em plural no JS devemos usar o numero entre os parenteses para indicar qual tag em especifico estamos se referindo.
