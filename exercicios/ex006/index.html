Condiçoes em JS

Desvio Condicional (Simples ou Dupla)

O Desvio Condicional serve para bifurcar o codigo, como exemplo eu posso fazer o seguinte:

        
            let n = 1
            n +=2
            alert('a variavel n vale ${n}')

No codigo de cima temos um codigo seguencial e o Desvio Condicional faz com que o codigo tenha dois caminhos para seguir como exemplo eu posso fazer com que o terceiro comando ocorra sem que o segundo seja executado fazendo a bifurcaçao.
Usa-se os comandos if para true e else para false, sendo que a prorpia condiçao e escirta da seguinte forma

        
            if(codiçao){
                o que vai ocorrer quando a condiçao for respeitada (true)
            } else{
                o que vai ocorrer se a condiçao n for respeitada (false)
            }
        

        Obs: A condiçao vale tanto para o if tanto para o else, sendo assim n precisa repetir a condiçao para o else e so ir direto

dentro dos blocos temos os comandos para cada resposta (true ou false)

Existem dois tipos de codiçoes basicas, que sao elas a condiçao simples que tem so o if que serve pra se a codiçao for respeitada aconteça algo e se n for respeitada n acontecera nada, e por fim a condiçao dupla que esta a cima, se a condiçao for respeitada vai ocorrer uma determinada coisa if se n acontecera outra coisa else
